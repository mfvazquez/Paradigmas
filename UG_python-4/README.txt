First lines of script ug.py configure its behaviour.

- ECI:
  Set to False if you want to run or test the script without an ECI device.

- TASK_REPETITIONS:
  Total number of "real" games. A pause will occur automatically after half
  this amount of games.

- FULLSCREEN:
  Set to False to run program inside a window.

- TOP_OFFER:
  Amount of money in play in each trial.

- eci_address:
  IP address of the device receiving events through ECI protocol.

- eci_port:
  Port for ECI connection.
